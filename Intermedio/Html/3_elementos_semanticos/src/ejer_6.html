JavaScript
Un lenguaje de programación ampliamente utilizado para agregar interactividad y dinamismo a las páginas web.
HTML
Lenguaje de marcado utilizado para crear y estructurar el contenido de las páginas web.
CSS
Lenguaje utilizado para dar estilo y diseño a las páginas web, permitiendo controlar la presentación visual de los elementos HTML.
Inteligencia Artificial
Campo de la informática que se ocupa de crear sistemas capaces de realizar tareas que normalmente requieren inteligencia humana.
Machine Learning
Subcampo de la inteligencia artificial que se enfoca en el desarrollo de algoritmos que permiten a las computadoras aprender a partir de datos y experiencias.
Blockchain
Tecnología de registro distribuido que proporciona una forma segura y transparente de realizar transacciones y almacenar datos de manera descentralizada.
